# 資料表說明

## data.db 資料庫結構

### `movies` 資料表
本資料表用於儲存從 [SSR1 Scrape Center](https://ssr1.scrape.center/) 抓取的電影資訊。

| 欄位名稱 | 資料型態 | 說明 | 範例 |
| :--- | :--- | :--- | :--- |
| `id` | INTEGER | 主鍵，自動遞增 | `1` |
| `title` | TEXT | 電影名稱 (包含中文與英文) | `霸王别姬 - Farewell My Concubine` |
| `score` | TEXT | 電影評分 | `9.5` |
| `categories` | TEXT | 電影類型 (多個類型以 `/` 分隔) | `剧情/爱情` |
| `country` | TEXT | 製作國家/地區 | `中国内地、中国香港` |
| `duration` | TEXT | 電影時長 | `171 分钟` |
| `release_date` | TEXT | 上映日期 (格式：YYYY-MM-DD，若無則顯示原始文字或空) | `1993-07-26` |
